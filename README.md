# lessi

Reduce pictures to a specified color palette.

## Example

The following:

'''
$ lessi -p '#A42864 #CE3661 #D9636D #E49990 #EFCCBD #FAF2EA #222222' -s -m 80 -o example.gif photo.jpg
'''
