# sassie

Apply specified color palette to an image and convert it to svg.

## Example

The following .jpg:

```
$ lessi -p '#A42864 #CE3661 #D9636D #E49990 #EFCCBD #FAF2EA #222222' -s -m 80 -o example.svg photo.jpg
```
